# 블로그 검토 리포트 (2026-04-20 ~ 2026-05-12)

검토 대상: 14편 블로그 본문 + SEO 메타 + 인포그래픽 샘플
검토 관점: 네이버 SEO/노출, 인포그래픽 품질, 가독성/체류시간, 콘텐츠 정확성
페르소나: 모바일 네이버 앱 점심시간 30-40대 직장인 (초보~중수)

## 잘 되고 있는 것
1. **4섹션 골조의 일관성** — 시장 분석 → 투자 심리 → 핵심 포인트 → 내 생각 → 적용할 점. 14편 내내 유지.
2. **숫자 디테일** — 단순 등락률이 아닌 수급/실적 수치(예: 05-12 외국인 1.1조 매도 vs 개인 9,428억 매수).
3. **심리 함정 → 교정법** 구조 — 'FOMO 추격', 'Buy the rumor, Sell the fact' 등.
4. **'게임 기획자' 비유 페르소나** — 차별화 + 체류시간 핵심 자산.
5. **본문 분량 안정** — 모두 공백포함 2,500~3,500자, 네이버 권장 충족.

## 약점 / 리스크 (우선순위)

### [치명] 1. 제목 너무 길고 키워드가 뒤로 밀림
- 05-12 제목 38자, 05-11 제목 40자 → 모바일 검색에서 잘림
- `[날짜]` 접두어가 클릭률 까먹는 주범
- **추정 CTR 손실 25~35%**

### [치명] 2. 인포그래픽 스타일 단절
- 04-27: 800×450 가로 SVG, 5종, 그라데이션
- 05-06 이후: 900×900 정방형 HTML, 4종, 단색
- 사용자 요청 사양(1600×900)과 실제 산출물 불일치
- 모바일에서 정방형 = 화면 가득 → 이탈 자극

### [심각] 3. market 카드 한글 잘림 리스크
- 38px stat-value에 19자 한글 → 강제 줄바꿈/넘침
- 원래 stat-card는 짧은 숫자 한 줄용

### [심각] 4. 첫 100자 후킹 부재
- 14편 모두 영상 정보 테이블이 검색 미리보기에 노출됨
- **추정 CTR 손실 15~20%**

### [중간] 5. 태그 30개 활용 부족
- 04-20 10개, 04-30 13개, 05-12 15개 — 네이버 권장 30개 미달
- 롱테일 키워드 비중 약함

### [중간] 6. 시계열 사실 일관성
- 호르무즈가 4월~5월 사이 4번 봉쇄/교전됐다 풀림
- 같은 사건이 매번 다르게 서술됨
- 블로그 내부 링크/시리즈화 부재

### [중간] 7. 5월 6일 6.7% 급등 서술의 비현실성
- 일일 6.7% = 사이드카 발동 수준인데 본문에 언급 없음
- 신뢰도 타격 가능

### [중간] 8. 영상 메타가 본문에 노출
- `(7:10~)`, "광수 생각", "오후장 한 마디" 등 영상 안 본 독자에게 산만

## 개선 액션 (Top 5)

**즉시 (현재 14편 일괄 수정)**
1. **제목 28~32자 + 키워드 앞쪽 배치**: `코스피 8,000 직전 반락 — 외국인 1.1조 매도가 말한 것` (날짜는 본문에)

**구조 개선**
