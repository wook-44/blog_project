# SEO 전략 에이전트 가이드

블로그 포스트가 완성된 뒤, 발행 플랫폼별 SEO를 최적화하는 에이전트.
각 플랫폼의 알고리즘과 독자 특성이 다르므로, 동일한 콘텐츠를 플랫폼에 맞게 변형한다.

---

## 대상 플랫폼

| 플랫폼 | 알고리즘 특성 | 주 독자층 |
|---|---|---|
| 네이버 블로그 | C-Rank + D.I.A. (직접성·정보성) | 30~50대 국내 투자자 |
| 티스토리 | 구글 SEO (백링크, 체류시간) | 2030 투자/재테크 관심층 |
| 브런치 | 에디터 큐레이션 + 공유 바이럴 | 글 읽는 문화 관심층 |
| 노션 퍼블리시 | 없음 (직접 공유 중심) | 지인/팔로워 |

---

## 플랫폼별 SEO 최적화 지침

### 1. 네이버 블로그

**알고리즘 핵심**: C-Rank(블로그 신뢰도) + D.I.A.(문서 직접성·정보성)
- 광고성/복사 콘텐츠를 강하게 패널티

**제목 전략**
- 30자 이내, 핵심 키워드 앞에 배치
- 검색어 그대로 포함: "나스닥 전망", "금리 동결 주식", "오늘 증시 분석"
- 숫자 포함 시 클릭률 상승: "3가지", "2025년", "+12.4%"
- 예시: `[2026.05.04] 나스닥 기술주 전망 — 금리 동결 이후 3가지 투자 전략`

**본문 최적화**
- 목차 태그 적극 활용 (소제목마다 H2/H3)
- 첫 문단에 핵심 키워드 2~3회 자연스럽게 포함
- 500자 이상 문단 분리 → 가독성 향상
- 이미지마다 alt 텍스트 = 키워드 포함 한 줄 설명
- 내부 링크: 같은 주제의 이전 글로 연결
- 마지막 문단: "공감과 댓글은 큰 힘이 됩니다" 형태의 CTA

**태그 전략**
- 태그 10~15개: [핵심 종목명] [시장 키워드] [날짜] [투자전략] 조합
- 예: `나스닥`, `금리동결`, `기술주전망`, `2026주식`, `반도체주식`

**발행 최적화**
- 발행 시간: 오전 7~9시 또는 오후 12~2시 (활성 사용자 피크)
- 이미지 최소 3장 이상 포함

---

### 2. 티스토리

**알고리즘 핵심**: 구글 검색 인덱싱 (구글 SEO 그대로 적용)

**제목 전략**
- 60자 이내 (구글 타이틀 제한)
- 검색 의도 매칭: "~하는 방법", "~란 무엇인가", "~전망 정리"
- 브랜드명 후미 배치: `나스닥 기술주 전망 2026 | 주식분석블로그`

**본문 최적화**
- 도입부 150자 이내에 핵심 키워드 포함 (구글 스니펫 최적화)
- 소제목(H2)마다 키워드 변형 포함
- 문단 길이: 3~5줄 (모바일 가독성)
- 내부 링크 2~3개 + 외부 참고 링크 1~2개
- 이미지 파일명 = 키워드 슬러그 (예: `nasdaq-tech-stock-2026.png`)
- 목차(TOC) 플러그인 활용

**메타 디스크립션**
- 150~160자: 키워드 포함 + 클릭 유도 문구
- 예: `2026년 5월 나스닥 기술주 전망을 12시에 만나요 영상을 바탕으로 분석했습니다. 금리 동결 이후 3가지 투자 전략과 주목 종목을 정리합니다.`

**카테고리/태그**
- 카테고리: `주식분석 > 시장동향` 형태의 계층 구조
- 태그: 5~7개 핵심만 (과다 태그 구글 패널티)

---

### 3. 브런치

**알고리즘 핵심**: 에디터 큐레이션 + 소셜 공유

**제목 전략**
- 감성적이고 호기심 자극하는 형태
- 질문형: "지금 기술주를 사도 될까요?"
- 역설형: "시장이 무서울 때 오히려 사야 하는 이유"
- 숫자 최소화, 공감 키워드 최대화

**본문 최적화**
- 첫 문단: 독자의 공감을 끄는 상황 묘사로 시작
- 에세이 스타일: 딱딱한 표 보다 서사적 흐름 선호
- 분량: 1,500~3,000자 (너무 짧으면 큐레이션 불리)
- 마지막: 독자에게 질문 던지기 → 댓글 유도

**발행 전략**
- 발행 후 트위터/X, 오픈카톡방에 공유
- 시리즈로 묶기: `12시에 만나요 분석 시리즈 #3`

## SEO 에이전트 실행 출력물

블로그 원본을 기반으로 각 플랫폼용 변형본을 생성한다.

```
seo_output: {
  "naver": {
    "title": "[최적화 제목]",
    "tags": ["태그1", "태그2", ...],
    "intro_paragraph": "[네이버용 첫 문단]",
    "publish_time": "07:30",
    "cta": "[공감/댓글 유도 문구]"
  },
  "tistory": {
    "title": "[SEO 제목]",
    "meta_description": "[150자 메타 디스크립션]",
    "slug": "yyyy-mm-dd-keyword-slug",
    "image_alt_texts": {
      "market": "[시장분석 이미지 alt]",
      "psychology": "[투자심리 이미지 alt]",
      "summary": "[핵심포인트 이미지 alt]",
      "insight": "[인사이트 이미지 alt]"
    },
    "tags": ["태그1", ...],
    "category": "주식분석/시장동향"
  },
  "brunch": {
    "title": "[감성 제목]",
    "subtitle": "[부제목]",
    "hook_paragraph": "[브런치용 도입부]",
    "series_tag": "12시에 만나요 분석 시리즈"
  }
}
```

## 공통 키워드 전략

매 포스트에서 다음 키워드 카테고리를 조합한다:

**시간 키워드** (검색 최신성): `2026년 5월`, `오늘 증시`, `이번 주 주식`

**이벤트 키워드** (영상 주제에서 추출): `금리 동결`, `나스닥 반등`, `반도체 섹터`

**행동 키워드** (검색 의도): `전망`, `분석`, `투자 전략`, `매수 타이밍`

**채널 키워드** (브랜드): `12시에 만나요`, `주식 유튜브 정리`

최종 키워드 조합 예시: `2026년 5월 나스닥 기술주 전망 투자 전략`
