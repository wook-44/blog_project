# 유튜브 영상 설명란 / 고정댓글 가이드 (v2 — 2026-05-18)

매일 발행되는 블로그를 **유튜브 외부 유입 채널**로 활용하기 위한 표준 템플릿.
이 가이드는 자동 파이프라인이 `TODAY_seo.md` 의 "유튜브 설명란용" 섹션을 생성할 때 따른다.

---

## 1. 왜 필요한가

- 유튜브 영상 시청자는 "오늘 어떤 종목 얘기였지?" "차트 다시 보고 싶다" 라는 후속 검색 수요가 있다.
- 영상 설명란/고정댓글에 블로그 링크가 일관되게 박혀 있으면, **시청자 → 블로그 전환률**이 올라간다.
- 네이버 검색 트래픽이 아닌 **유튜브 인앱 클릭**이라 별개의 유입 채널이 된다.

---

## 2. 영상 설명란용 한 줄 (필수)

발행 후 영상 설명란 **상단 1~2번째 줄**에 붙여넣을 한 줄.

```
📊 오늘 영상 풀 분석 + 인포그래픽 → [블로그 제목] | [블로그 URL]
```

### 예시
```
📊 오늘 영상 풀 분석 + 인포그래픽 → 코스피 6,512 상승 마감 — 5월 18일 외국인 1.2조 매수 전환
👉 https://wook-44.github.io/blog_project/daily/2026-05-18/
```

### 룰
- **이모지 1개만** (📊 또는 📈). 과다 사용 금지.
- **블로그 제목 그대로** — SEO 키워드 일치로 유튜브 알고리즘이 관련성 인식.
- **URL은 GitHub Pages 또는 네이버 블로그 단축 URL** 중 하나로 통일 (이 가이드 §5 참조).

---

## 3. 고정댓글용 3줄 템플릿

영상 게시 후 고정댓글로 박을 좀 더 풍부한 버전.

```
📊 오늘 영상 풀 분석 정리했어요.
- 🔗 관련 종목 한눈에 표 + 인포그래픽 3장
- 📌 영상 핵심 발언 타임스탬프 모음

👉 [블로그 URL]
```

### 룰
- **3줄 이내** (모바일 댓글창에서 잘리지 않는 길이).
- 블로그가 영상에 없는 부가 가치를 준다는 점 강조 — "관련 종목 표", "인포그래픽", "타임스탬프" 등.
- 마지막 줄은 URL만 단독 배치 (자동 링크화).

---

## 4. 자동 생성 흐름

`daily-stock-blog-writer` 에이전트 6(SEO 전략) 단계에서 `TODAY_seo.md` 마지막에 다음 블록을 자동 생성:

```markdown
## [유튜브 영상 설명란용]

### 한 줄 (설명란 상단용)
📊 오늘 영상 풀 분석 + 인포그래픽 → {title}
👉 {blog_url}

### 3줄 (고정댓글용)
📊 오늘 영상 풀 분석 정리했어요.
- 🔗 관련 종목 한눈에 표 + 인포그래픽 {N}장
- 📌 영상 핵심 발언 타임스탬프 모음

👉 {blog_url}
```

`copy_tool.html` 의 8번 시안색 버튼이 이 두 블록을 한 번에 복사할 수 있게 한다.

## 5. 블로그 URL 표준

3가지 옵션 중 **GitHub Pages** 를 1순위로 권장.

| 옵션 | URL 형식 | 장단점 |
|---|---|---|
| GitHub Pages | `https://wook-44.github.io/blog_project/daily/YYYY-MM-DD/` | 안정적, 인덱싱 잘됨, 본문·이미지 모두 노출. 다만 GitHub Pages 활성화 필요. |
| 네이버 블로그 단축 URL | `https://blog.naver.com/{id}/{postno}` | 네이버 검색 점수 ↑. 단축 URL은 수동 복사 필요. |
| 티스토리 URL | `https://{id}.tistory.com/{slug}` | 검색 노출 OK, 부가 트래픽 채널. |

**전환 흐름** (가장 트래픽 효율 높은 순):
1. 유튜브 → GitHub Pages (즉시 보기, 광고 없음)
2. GitHub Pages 본문 끝에 네이버 블로그 원본 링크 → 네이버 추가 트래픽

GitHub Pages 활성화 여부는 `wook-44/blog_project` 저장소 Settings → Pages 에서 확인.
미활성 시 우선 네이버 블로그 단축 URL 사용, 활성화 작업은 별도 처리.

## 6. 수동 작업 순서 (주인님)

매일 17:30 본편 발행 완료 알림(텔레그램)이 오면:

1. 텔레그램 알림에서 블로그 제목 확인
2. `copy_tool.html` 의 **8번(시안) 버튼** 클릭 → 클립보드에 한 줄 + 3줄 복사
3. 유튜브 스튜디오 → 해당 영상 → 설명 편집 → **상단 1~2번째 줄에 한 줄 붙여넣기** → 저장
4. 같은 영상 → 댓글 영역 → 새 댓글에 **3줄 붙여넣기** → 게시 → 점 메뉴 → **댓글 고정**

소요 시간: 약 1~2분.

---

## 7. 측정 (선택)

유튜브 → 블로그 전환 트래픽을 측정하려면 URL 끝에 UTM 파라미터 추가:
```
?utm_source=youtube&utm_medium=description&utm_campaign=12time_{YYYYMMDD}
```

GitHub Pages는 GA 연결이 필요. 네이버 블로그는 자체 통계의 "유입 경로 → youtube.com" 으로 확인.
초기엔 측정 없이 **3주 운영 후 트래픽 변화**만 비교해도 효과 확인 가능.
